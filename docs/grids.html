##Easy columns Content can be gridded off into one to six columns simply by added a 'columned' class to the parent. The children can be any element at all: divs, list items, sections... anything you like. foo class="columned"
##Controllable... By default, content will be display in three columns. However, you can set the number by adding a class specifying the number of columns: 'four-col', 'two-col', 'five-col', etc: foo class="columned four-col"
##Or automatic... There is some support for 'columned auto', to set the number of columns according to available sensible width. In fact, this column grid is set to auto width: foo class="columned auto"

##Responsive On small screens, multiple columns will be reset to a more appropriate width. For example, four-col will automatically collapse down to two-col on very small screens.
##Layout-ready There is also support for 'columned phi' and 'columned phi reverse', which both automatically break content into two columns based on the golden ratio.
##Nestable You can nest grids as much as you want. Need four columns in your main area and two in your side bar? No problem.
