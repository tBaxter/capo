--- layout: default title: Capo typography --- ### Typography Capo takes your typography seriously. It defines a solid typographic scale and baseline grid, and ensures elements fall on it. Best of all, it makes good typography easy, with simple variables and mixins. You mark up you code semantically, and know the styles are right.

This is the fine print # The details Capo typography is based on a 10px baseline grid. All type measurements are determined by the baseline and are relative to it. Examples of various components and elements can be seen falling on the baseline grid on this page. - Top-level headings (H1) are 4x the baseline grid. - Second-level headings (H2) are 3x the baseline grid. - Other elements are 2x the baseline grid. - Fine print is 75% of the baseline grid and set smaller than the default font size. # Choosing the correct tag Do not choose tags based on the default size, choose them semantically. You can apply the mixin you need for visual style.
